# Blueprint: 행운의 로또 번호 생성기

## 개요

이 애플리케이션은 사용자에게 1부터 45까지의 숫자 중 6개의 고유한 로또 번호를 생성해주는 간단하고 시각적으로 매력적인 웹 애플리케이션입니다.

## 기능

- **번호 생성:** '번호 생성' 버튼을 클릭하면 6개의 고유한 랜덤 번호가 생성됩니다.
- **자동 정렬:** 생성된 번호는 오름차순으로 자동 정렬되어 표시됩니다.
- **초기 번호 로드:** 페이지가 처음 로드될 때 한 세트의 번호가 자동으로 생성됩니다.
- **애니메이션:** 번호가 나타날 때 부드러운 'pop-in' 애니메이션 효과가 적용됩니다.

## 디자인 및 스타일

- **모던 디자인:** 그림자 효과와 둥근 모서리를 사용하여 깔끔하고 현대적인 UI를 구현했습니다.
- **배경 질감:** 배경에 미묘한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
- **색상 팔레트:** `oklch` 색상 공간을 사용하여 생동감 있고 접근성 높은 색상을 사용했습니다. 각 번호 범위에 따라 다른 색상이 지정됩니다.
    - 1-10: 노란색 계열
    - 11-20: 파란색 계열
    - 21-30: 빨간색 계열
    - 31-40: 회색 계열
    - 41-45: 녹색 계열
- **타이포그래피:** 가독성이 좋은 'Nanum Gothic' 웹 폰트를 사용했습니다.
- **인터랙티브 요소:** 버튼에는 마우스 호버 및 클릭 시 시각적 피드백(그림자, 아이콘 회전)이 포함되어 있습니다.
- **반응형 디자인:** 다양한 화면 크기에서 잘 보이도록 설계되었습니다.

## 현재 구현 계획

1.  **HTML (`index.html`):** 프로젝트의 기본 구조를 설정합니다.
    -   페이지 제목, 헤더, 번호가 표시될 컨테이너, 그리고 번호 생성 버튼을 포함합니다.
    -   Google Fonts에서 'Nanum Gothic' 폰트를 가져옵니다.
2.  **CSS (`style.css`):** 애플리케이션의 시각적 디자인을 담당합니다.
    -   CSS 변수(`:root`)를 사용하여 색상, 그림자, 폰트를 중앙에서 관리합니다.
    -   번호 공, 컨테이너, 버튼에 대한 스타일을 정의합니다.
    -   번호가 나타나는 애니메이션(`@keyframes pop-in`)을 추가합니다.
    -   번호 범위에 따른 색상 클래스(`number-color-*`)를 정의합니다.
3.  **JavaScript (`main.js`):** 애플리케이션의 핵심 로직을 구현합니다.
    -   버튼 클릭 이벤트를 감지합니다.
    -   6개의 고유한 난수를 생성하고 정렬합니다.
    -   DOM 요소를 동적으로 생성하여 번호를 화면에 표시합니다.
    -   각 번호에 적절한 색상 클래스를 할당합니다.
4.  **문서 (`blueprint.md`):** 프로젝트의 목적, 기능, 디자인을 문서화합니다.
