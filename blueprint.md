# Blueprint: 행운의 로또 번호 생성기

## 개요

이 애플리케이션은 사용자에게 1부터 45까지의 숫자 중 6개의 고유한 로또 번호를 생성해주는 간단하고 시각적으로 매력적인 웹 애플리케이션입니다. 다크 모드와 라이트 모드를 지원합니다.

## 기능

- **번호 생성:** '번호 생성' 버튼을 클릭하면 6개의 고유한 랜덤 번호가 생성됩니다.
- **자동 정렬:** 생성된 번호는 오름차순으로 자동 정렬되어 표시됩니다.
- **초기 번호 로드:** 페이지가 처음 로드될 때 한 세트의 번호가 자동으로 생성됩니다.
- **애니메이션:** 번호가 나타날 때 부드러운 'pop-in' 애니메이션 효과가 적용됩니다.
- **테마 전환:** 사용자는 라이트 모드와 다크 모드 간에 테마를 전환할 수 있습니다. 사용자의 선택은 브라우저에 저장됩니다.

## 디자인 및 스타일

- **모던 디자인:** 그림자 효과와 둥근 모서리를 사용하여 깔끔하고 현대적인 UI를 구현했습니다.
- **배경 질감:** 배경에 미묘한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
- **색상 팔레트:** `oklch` 색상 공간을 사용하여 생동감 있고 접근성 높은 색상을 사용했습니다. 각 번호 범위에 따라 다른 색상이 지정됩니다.
- **타이포그래피:** 가독성이 좋은 'Nanum Gothic' 웹 폰트를 사용했습니다.
- **인터랙티브 요소:** 버튼과 토글 스위치에는 시각적 피드백이 포함되어 있습니다.
- **반응형 디자인:** 다양한 화면 크기에서 잘 보이도록 설계되었습니다.

## 현재 구현 계획

1.  **HTML (`index.html`):**
    -   테마 전환을 위한 토글 스위치를 추가합니다.
2.  **CSS (`style.css`):**
    -   라이트 및 다크 모드에 대한 색상 변수를 정의합니다.
    -   `data-theme` 속성을 사용하여 테마를 전환하는 스타일을 구현합니다.
    -   테마 토글 스위치의 스타일을 지정합니다.
3.  **JavaScript (`main.js`):**
    -   테마 토글 스위치의 상태 변경을 감지합니다.
    -   `<html>` 요소의 `data-theme` 속성을 변경하여 테마를 적용합니다.
    -   `localStorage`를 사용하여 사용자의 테마 설정을 저장하고 로드합니다.
    -   사용자의 시스템 환경 설정을 감지하여 초기 테마를 설정합니다.
4.  **Git 배포:**
    -   모든 변경 사항을 커밋하고 원격 GitHub 저장소에 푸시합니다.
